# QUB-RL

Humanoid Robot Reinforcement Learning Project using Isaac Gym.

## Robot Model

* QUB Humanoid Robot
* GPU optimized URDF
* Visual meshes and collision meshes included

## Directory Structure

assets/qub/
├── urdf/
└── meshes/

## Simulation

* Isaac Gym
* PPO based locomotion training

## Status

* Robot model imported
* Environment setup complete
* Walking policy under development
* QUB humanoid robot running in Isaac Gym parallel simulation environment.*
